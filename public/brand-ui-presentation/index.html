About project

Creative Direction

T. Hendo Dreamstation positions T. Hendo as a music creator inside a digital, game-inspired universe. The interface should feel like a sound station in deep space: dark, clean, premium, animated, and powered by neon color changes.

The product experience focuses on previewing tracks, exploring beats, purchasing licenses, signing into a personal dashboard, viewing purchases, and managing music through an admin panel.

Colors used

Color

The brand uses a black galaxy base, transparent glass panels, white text, and a six-color neon gradient. The same gradient powers headings, buttons, prices, progress bars, active rings, icons, and highlights.

The animated color system is fast because the palette is defined once as a static repeating CSS gradient. The UI only moves the gradient position, so the browser can animate the visual theme without constantly recalculating every color.
